## 🚀 Inicio Rápido

Sigue estos pasos para tener una copia local del proyecto en funcionamiento.

### 📋 Prerrequisitos

Necesitarás tener instalado:

* [Ej: Node.js (vX.X.X)]
* [Ej: npm o yarn]
* [Ej: Docker (opcional, si se usa)]

### 💻 Instalación

1.  **Clona el repositorio:**
    ```bash
    git clone [https://aws.amazon.com/es/what-is/repo/](https://aws.amazon.com/es/what-is/repo/)
    cd [nombre-del-proyecto]
    ```

2.  **Configura el Backend:**
    ```bash
    cd backend
    npm install  # o el comando de tu gestor de paquetes
    # Configura las variables de entorno (consulta CONFIGURACION/entorno_desarrollo)
    npm run dev  # o el comando de inicio
    ```

3.  **Configura el Frontend:**
    ```bash
    cd ../frontend
    npm install  # o el comando de tu gestor de paquetes
    npm start  # o el comando de inicio
    ```
    El frontend estará accesible en `http://localhost:[Puerto]`.
